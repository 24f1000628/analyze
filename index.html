The CI/CD Workflow

Our project incorporates a robust GitHub Actions workflow that automates the entire data processing and publishing cycle. Every push to the main branch triggers a series of steps:

  • ✅Code linting with Ruff for quality assurance.
  • ✅Execution of the Python data processing script.
  • ✅Publishing the generated result.json to GitHub Pages.

This ensures that the latest data insights are always available and that the codebase remains clean and functional.
